# 示例配置

虚构数据，用于测试脱敏效果。复制内容到 [`index.html`](../index.html) 试用。

| 文件 | 格式 | 主要敏感项 |
|------|------|-----------|
| `baota-nginx.conf` | Nginx / 宝塔 | 域名、内网 IP、证书路径 |
| `env-file.env` | 环境变量 | 数据库密码、云 AK/SK、JWT |
| `docker-compose.yml` | Docker Compose | 镜像、环境变量、卷路径 |
| `private-key.txt` | 私钥 | RSA / OpenSSH / EC 私钥块 |

使用步骤见 [介绍文章](../docs/zh/intro-article.md)。

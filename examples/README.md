# Sample configs

Fictional data for testing the masker. Copy into [`index.html`](../index.html).

| File | Format | Sensitive fields |
|------|--------|------------------|
| `baota-nginx.conf` | Nginx / Baota panel | Domains, private IPs, cert paths |
| `env-file.env` | Environment variables | DB passwords, cloud AK/SK, JWT |
| `docker-compose.yml` | Docker Compose | Images, env vars, volume paths |
| `private-key.txt` | Private keys | RSA / OpenSSH / EC key blocks |
